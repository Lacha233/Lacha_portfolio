Overview

Deployment Node.js app and redis using EC2 instances This Node.js code contain a basic user data for dashboard and account create,update,delete access. User can access once the URL request will go via VPC network and content serve by S3 bucket via cloud front. The whole process running in EC2 servers

Process

  • Create a VPC network using specific IP
  • Create and configure EC2 launch templates for App,Redis and NAT
  • Configure a VPC subnet for App,Redis and NAT using VPC network
  • Create VPC internetgateway to access the internet
  • Create S3 bucket and store Static files
  • Create a cloud front origin access identity to access S3 bucket.
  • Setup EBS and ELB for storage and control network traffic purpose
  • Launch the App,Redis and NAT EC2 instance

    • Access App using HTTP URLs

  • http://localhost:8080/health
  • http://localhost:8080/dashboard
  • http://localhost:8080/account/create?id=1&name=abc
  • http://localhost:8080/account/delete?id=2
